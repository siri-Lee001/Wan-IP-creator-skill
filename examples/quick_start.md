# 快速开始指南 (双模式版)

## 🚀 5分钟上手智能协作式IP创建

### 两种模式任选

#### 模式A: 从零创建 (全新设计)
**触发关键词**: "创建"、"设计"、"制作"、"新IP"
```
"我想创建一个赛博朋克风格的猫娘IP"

**触发关键词**: "已有"、"现有"、"上传"、"优化"

### 模式A: 从零创建流程
```
创意输入 → 智能提问 → 角色生成 → 用户确认 → 全自动7步

## 🆘 常见问题

### Q: 应该选择哪种模式？
A: 
- 有清晰创意但没图片 → 模式A
- 已有角色图片 → 模式B
- 有图片但想大改 → 模式A (重新设计)
- 有图片只想扩展 → 模式B (快速优化)

### Q: 图片质量要求？
A: 模式B建议:
- 格式: PNG/JPG/WebP
- 大小: > 1MB (建议)
- 分辨率: > 1024x1024
- 内容: 清晰角色，特征明显

### Q: 生成需要多长时间？
A: 
- 模式A完整流程: 20-35分钟
- 模式B完整流程: 15-25分钟
- 快速测试: 5-10分钟

### Q: 支持批量处理吗？
A: 支持！
- 多个角色批量分析
- 多个风格批量生成
- 多个产品批量设计

### Q: 可以中途切换模式吗？
A: 可以！随时提出调整需求。
